# MaskedCondition

The **MaskedCondition** column on a container field checks the value of another input field on the page to see if this field is displayed to the user.  Supports Boolean logic ands (&&), ors (||) and parenthesis.  If the check returns false, the input is not displayed to the user.

Identical to [VisibilityCondition](VisibilityCondition.md), but instead of not displaying the item, masks the item and shows white space where the input field would be.
