# Href

The **Href** column on a container field defines the URL for a [HyperLink](../05.Control-Details/HyperLink.md) control. It can include references to other container fields in the format `{{<FieldName>}}` where `<FieldName>` is the name of another field on the container.

## Controls

The **Href** column applies to the following controls:

- [HyperLink](../05.Control-Details/HyperLink.md)
